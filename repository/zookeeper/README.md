# Zookeeper Operator

The KUDO Zookeeper operator creates, configures and manages [Apache Zookeeper](https://zookeeper.apache.org/) clusters running on Kubernetes.

## Getting started

The latest stable version of Zookeeper operator is `0.3.1`
